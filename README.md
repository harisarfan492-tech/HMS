# HMS
A Hospital Management System built within C++, which will effectively allow a Patient, Doctor and Admin to inter-connect with each other effectively each other in a Hosptial environment


The entire repository consists of the backend of the code and the front end written within SFML 3.0.2. The working of the backend is primarily on the console and is quite simple but rather effective, the program actively guides either the Patient, admin or Doctor in using the program with interactable menu in which they can enter their choice on what to do
The GUI of the Program is a hybrid between the SFML window and the console window, it allows the user to select their desired option on the window but the entry of the data is done on the console, since implementing the logic and functionality was not feasible in the liimited time frame that I had, 
The GUI although functions just fine, some the functionaliies are not working properly which i am aware of and aim to correct them.
